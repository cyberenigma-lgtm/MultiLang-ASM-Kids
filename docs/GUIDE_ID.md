## Welcome!
Learn to speak to your computer in Indonesian! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `taruh` | Put a number in a box |
| **Add** | `tambah` | Add two numbers together |
| **Take** | `kurang` | Take away a number |
| **Show** | `tampil` | Show the result on screen |

## Your First Program

```masm
; Indonesian Program
taruh rax 5    ; Box 1 has 5
tambah rax 2    ; Add 2 (Now it's 7)
tampil rax     ; Show me the 7!
```
