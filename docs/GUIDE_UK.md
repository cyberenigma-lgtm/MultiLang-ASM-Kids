## Welcome!
Learn to speak to your computer in Ukrainian! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `polozhy` | Put a number in a box |
| **Add** | `dodaty` | Add two numbers together |
| **Take** | `vidnyaty` | Take away a number |
| **Show** | `pokazhy` | Show the result on screen |

## Your First Program

```masm
; Ukrainian Program
polozhy rax 5    ; Box 1 has 5
dodaty rax 2    ; Add 2 (Now it's 7)
pokazhy rax     ; Show me the 7!
```
