## Welcome!
Learn to speak to your computer in English! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `put` | Put a number in a box |
| **Add** | `add` | Add two numbers together |
| **Take** | `take` | Take away a number |
| **Show** | `look` | Show the result on screen |

## Your First Program

```masm
; English Program
put rax 5    ; Box 1 has 5
add rax 2    ; Add 2 (Now it's 7)
look rax     ; Show me the 7!
```
