## Welcome!
Learn to speak to your computer in Romanian! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `pune` | Put a number in a box |
| **Add** | `adauga` | Add two numbers together |
| **Take** | `scade` | Take away a number |
| **Show** | `arata` | Show the result on screen |

## Your First Program

```masm
; Romanian Program
pune rax 5    ; Box 1 has 5
adauga rax 2    ; Add 2 (Now it's 7)
arata rax     ; Show me the 7!
```
