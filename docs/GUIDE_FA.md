## Welcome!
Learn to speak to your computer in Persian! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `bezor` | Put a number in a box |
| **Add** | `jam` | Add two numbers together |
| **Take** | `kam` | Take away a number |
| **Show** | `neshon` | Show the result on screen |

## Your First Program

```masm
; Persian Program
bezor rax 5    ; Box 1 has 5
jam rax 2    ; Add 2 (Now it's 7)
neshon rax     ; Show me the 7!
```
