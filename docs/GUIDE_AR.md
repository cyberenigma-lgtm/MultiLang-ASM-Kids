## Welcome!
Learn to speak to your computer in Arabic! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `da` | Put a number in a box |
| **Add** | `ijma` | Add two numbers together |
| **Take** | `itrah` | Take away a number |
| **Show** | `anzur` | Show the result on screen |

## Your First Program

```masm
; Arabic Program
da rax 5    ; Box 1 has 5
ijma rax 2    ; Add 2 (Now it's 7)
anzur rax     ; Show me the 7!
```
