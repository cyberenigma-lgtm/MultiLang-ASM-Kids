## Welcome!
Learn to speak to your computer in Dutch! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `zet` | Put a number in a box |
| **Add** | `tel_op` | Add two numbers together |
| **Take** | `trek_af` | Take away a number |
| **Show** | `toon` | Show the result on screen |

## Your First Program

```masm
; Dutch Program
zet rax 5    ; Box 1 has 5
tel_op rax 2    ; Add 2 (Now it's 7)
toon rax     ; Show me the 7!
```
