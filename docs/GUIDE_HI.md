## Welcome!
Learn to speak to your computer in Hindi! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `rakho` | Put a number in a box |
| **Add** | `jodo` | Add two numbers together |
| **Take** | `ghatao` | Take away a number |
| **Show** | `dikhao` | Show the result on screen |

## Your First Program

```masm
; Hindi Program
rakho rax 5    ; Box 1 has 5
jodo rax 2    ; Add 2 (Now it's 7)
dikhao rax     ; Show me the 7!
```
