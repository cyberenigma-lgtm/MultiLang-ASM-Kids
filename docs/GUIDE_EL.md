## Welcome!
Learn to speak to your computer in Greek! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `bale` | Put a number in a box |
| **Add** | `prosthese` | Add two numbers together |
| **Take** | `afairese` | Take away a number |
| **Show** | `diekse` | Show the result on screen |

## Your First Program

```masm
; Greek Program
bale rax 5    ; Box 1 has 5
prosthese rax 2    ; Add 2 (Now it's 7)
diekse rax     ; Show me the 7!
```
