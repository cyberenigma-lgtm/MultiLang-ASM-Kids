## Welcome!
Learn to speak to your computer in Korean! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `neoh` | Put a number in a box |
| **Add** | `deohae` | Add two numbers together |
| **Take** | `ppaera` | Take away a number |
| **Show** | `boyeo` | Show the result on screen |

## Your First Program

```masm
; Korean Program
neoh rax 5    ; Box 1 has 5
deohae rax 2    ; Add 2 (Now it's 7)
boyeo rax     ; Show me the 7!
```
