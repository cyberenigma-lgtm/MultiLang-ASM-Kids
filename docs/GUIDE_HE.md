## Welcome!
Learn to speak to your computer in Hebrew! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `sim` | Put a number in a box |
| **Add** | `hosef` | Add two numbers together |
| **Take** | `haser` | Take away a number |
| **Show** | `hare` | Show the result on screen |

## Your First Program

```masm
; Hebrew Program
sim rax 5    ; Box 1 has 5
hosef rax 2    ; Add 2 (Now it's 7)
hare rax     ; Show me the 7!
```
