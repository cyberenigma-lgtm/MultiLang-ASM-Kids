## Welcome!
Learn to speak to your computer in Vietnamese! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `dat` | Put a number in a box |
| **Add** | `them` | Add two numbers together |
| **Take** | `tru` | Take away a number |
| **Show** | `hien` | Show the result on screen |

## Your First Program

```masm
; Vietnamese Program
dat rax 5    ; Box 1 has 5
them rax 2    ; Add 2 (Now it's 7)
hien rax     ; Show me the 7!
```
