## Welcome!
Learn to speak to your computer in Malay! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `letak` | Put a number in a box |
| **Add** | `tambah` | Add two numbers together |
| **Take** | `tolak` | Take away a number |
| **Show** | `tunjuk` | Show the result on screen |

## Your First Program

```masm
; Malay Program
letak rax 5    ; Box 1 has 5
tambah rax 2    ; Add 2 (Now it's 7)
tunjuk rax     ; Show me the 7!
```
