## Welcome!
Learn to speak to your computer in Thai! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `sai` | Put a number in a box |
| **Add** | `buak` | Add two numbers together |
| **Take** | `lop` | Take away a number |
| **Show** | `sadang` | Show the result on screen |

## Your First Program

```masm
; Thai Program
sai rax 5    ; Box 1 has 5
buak rax 2    ; Add 2 (Now it's 7)
sadang rax     ; Show me the 7!
```
