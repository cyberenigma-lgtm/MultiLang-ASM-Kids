## Welcome!
Learn to speak to your computer in Polish! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `wsadz` | Put a number in a box |
| **Add** | `dodaj` | Add two numbers together |
| **Take** | `odejmij` | Take away a number |
| **Show** | `pokaz` | Show the result on screen |

## Your First Program

```masm
; Polish Program
wsadz rax 5    ; Box 1 has 5
dodaj rax 2    ; Add 2 (Now it's 7)
pokaz rax     ; Show me the 7!
```
