## Welcome!
Learn to speak to your computer in Japanese! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `irete` | Put a number in a box |
| **Add** | `tashite` | Add two numbers together |
| **Take** | `hiite` | Take away a number |
| **Show** | `misete` | Show the result on screen |

## Your First Program

```masm
; Japanese Program
irete rax 5    ; Box 1 has 5
tashite rax 2    ; Add 2 (Now it's 7)
misete rax     ; Show me the 7!
```
