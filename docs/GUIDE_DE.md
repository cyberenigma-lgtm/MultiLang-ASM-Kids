## Welcome!
Learn to speak to your computer in Deutsch! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `setze` | Put a number in a box |
| **Add** | `addiere` | Add two numbers together |
| **Take** | `ziehe_ab` | Take away a number |
| **Show** | `zeige` | Show the result on screen |

## Your First Program

```masm
; Deutsch Program
setze rax 5    ; Box 1 has 5
addiere rax 2    ; Add 2 (Now it's 7)
zeige rax     ; Show me the 7!
```
