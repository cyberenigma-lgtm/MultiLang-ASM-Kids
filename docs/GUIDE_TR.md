## Welcome!
Learn to speak to your computer in Turkish! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `koy` | Put a number in a box |
| **Add** | `ekle` | Add two numbers together |
| **Take** | `cikar` | Take away a number |
| **Show** | `goster` | Show the result on screen |

## Your First Program

```masm
; Turkish Program
koy rax 5    ; Box 1 has 5
ekle rax 2    ; Add 2 (Now it's 7)
goster rax     ; Show me the 7!
```
