## Welcome!
Learn to speak to your computer in Tagalog! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `lagay` | Put a number in a box |
| **Add** | `dagdag` | Add two numbers together |
| **Take** | `bawas` | Take away a number |
| **Show** | `pakita` | Show the result on screen |

## Your First Program

```masm
; Tagalog Program
lagay rax 5    ; Box 1 has 5
dagdag rax 2    ; Add 2 (Now it's 7)
pakita rax     ; Show me the 7!
```
