## Welcome!
Learn to speak to your computer in Italiano! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `metti` | Put a number in a box |
| **Add** | `aggiungi` | Add two numbers together |
| **Take** | `togli` | Take away a number |
| **Show** | `mostra` | Show the result on screen |

## Your First Program

```masm
; Italiano Program
metti rax 5    ; Box 1 has 5
aggiungi rax 2    ; Add 2 (Now it's 7)
mostra rax     ; Show me the 7!
```
