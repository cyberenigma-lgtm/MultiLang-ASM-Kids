## Welcome!
Learn to speak to your computer in Chinese! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `fang` | Put a number in a box |
| **Add** | `jia` | Add two numbers together |
| **Take** | `jian` | Take away a number |
| **Show** | `kan` | Show the result on screen |

## Your First Program

```masm
; Chinese Program
fang rax 5    ; Box 1 has 5
jia rax 2    ; Add 2 (Now it's 7)
kan rax     ; Show me the 7!
```
