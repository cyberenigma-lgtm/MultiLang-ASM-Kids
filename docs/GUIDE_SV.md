## Welcome!
Learn to speak to your computer in Swedish! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `stall` | Put a number in a box |
| **Add** | `addera` | Add two numbers together |
| **Take** | `dra_av` | Take away a number |
| **Show** | `visa` | Show the result on screen |

## Your First Program

```masm
; Swedish Program
stall rax 5    ; Box 1 has 5
addera rax 2    ; Add 2 (Now it's 7)
visa rax     ; Show me the 7!
```
