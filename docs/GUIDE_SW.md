## Welcome!
Learn to speak to your computer in Swahili! 

In MultiLang-ASM Kids, we use magic words to move numbers around.

## The Magic Words (Keywords)

| Action | Command | Meaning |
| :--- | :--- | :--- |
| **Put** | `weka` | Put a number in a box |
| **Add** | `ongeza` | Add two numbers together |
| **Take** | `toa` | Take away a number |
| **Show** | `onyesha` | Show the result on screen |

## Your First Program

```masm
; Swahili Program
weka rax 5    ; Box 1 has 5
ongeza rax 2    ; Add 2 (Now it's 7)
onyesha rax     ; Show me the 7!
```
